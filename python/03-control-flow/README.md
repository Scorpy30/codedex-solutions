# Key Learnings

- Lists use indexes while dictionaries use keys
- Dictionary keys are case-sensitive
- Duplicate dictionary keys overwrite previous values
- Consistent dictionary schemas make looping scalable
- Nested data structures allow real-world style modeling
- `.items()` helps iterate through dictionary key-value pairs
- Dynamic score tracking can be implemented cleanly using dictionaries
- Structuring data properly simplifies logic significantly
